# 更新日志 - v1.0.1

## 修复内容

### 🐛 Bug 修复

1. **修复图片发送问题**
   - 使用 `Plain` 组件包装文本消息，确保 `chain_result` 正常工作
   - 添加详细的日志记录，便于调试
   - 改进错误处理，所有图片生成和发送都包裹在 try-except 中

2. **改进错误处理**
   - 生成游戏失败时，显示错误信息并清理游戏状态
   - 生成下一题失败时，提示用户重新开始游戏
   - 删除临时图片失败时，记录警告日志但不影响游戏继续

3. **添加日志记录**
   - 游戏生成过程的每一步都有日志记录
   - 便于排查问题和调试

## 变更详情

### 代码修改

**main.py:**
- 导入 `Plain` 组件：`from astrbot.api.message_components import Image, Plain`
- 所有 `chain_result` 调用中的文本部分都用 `Plain()` 包装
- 添加 `logger.info()` 和 `logger.error()` 日志记录
- 所有图片生成和发送代码都用 try-except 包裹

### 修复前的问题

```python
# 修复前 - 可能导致消息发送失败
yield event.chain_result([
    Image.fromFileSystem(str(img_path)),
    "文本消息"  # ❌ 裸字符串
])
```

### 修复后

```python
# 修复后 - 正确使用消息组件
yield event.chain_result([
    Image.fromFileSystem(str(img_path)),
    Plain("文本消息")  # ✅ 使用 Plain 组件
])
```
