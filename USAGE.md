# 清一色听牌训练插件 - 使用指南

## 功能概述

这是一个用于 AstrBot 的日本麻将清一色听牌训练插件。插件会随机生成13张同花色的待听牌型，玩家需要判断听哪些牌。

## 游戏规则

### 基本规则

1. **抢答制**：一个群共享一个游戏，多人可同时参与抢答
2. **完整作答**：必须回答出所有听牌才算正确（顺序不限）
3. **连胜记录**：每个用户独立记录连胜次数
4. **错误处理**：
   - 答错仅提示"错误"，不显示正确答案
   - 同一用户连续错误5次后，显示正确答案并继续下一题
   - 5分钟内无人作答，显示正确答案并停止游戏

### 花色说明

- `m` = 万（Man）
- `s` = 索（Sou）
- `p` = 饼（Pin）

## 命令使用

### 开始游戏

```
/清一色
```

BOT 会生成一副13张牌的待听牌型图片，并告知花色。

### 作答

```
/清一色 1m2m3m
```

**格式说明：**
- 数字在前，花色在后
- 可以连写多张牌，如 `1m2m` 表示听1万和2万
- 也可以写成 `12m`，表示听1万和2万
- 必须使用当前题目的花色

**示例：**
- `/清一色 1m` - 听1万
- `/清一色 1m4m` 或 `/清一色 14m` - 听1万和4万
- `/清一色 2s5s8s` 或 `/清一色 258s` - 听2索、5索、8索

### 停止游戏

```
/清一色 stop
```

```
用户A: /清一色
```

## 安装说明

### 依赖安装

```bash
pip install Pillow
```

或者使用 requirements.txt：

```bash
pip install -r requirements.txt
```

### 插件配置

将插件文件夹放入 AstrBot 的插件目录，重启 AstrBot 即可自动加载。

## 技术说明

### 和牌判定

- 使用标准日本麻将规则：4组面子 + 1对雀头
- 面子包括：顺子（3张连续）、刻子（3张相同）
- 自动验证生成的牌型必须有听牌

### 图片生成

- 使用 `Regular` 文件夹中的麻将牌资源
- 自动将花色图层叠加到底牌上
- 13张牌横向排列，间距2像素

### 游戏状态

- 群级别：每个群/频道共享一个游戏状态
- 用户级别：每个用户独立记录连胜次数和错误次数
- 超时检测：5分钟无人作答自动显示答案并停止

## 常见问题

**Q: 为什么我答对了一张还说错误？**

A: 必须回答出所有听牌才算正确。比如听1m、4m、7m，你只回答"1m"是不对的，必须回答"1m4m7m"或"147m"。

**Q: 答案顺序重要吗？**

A: 不重要。"1m4m7m"、"147m"、"741m"、"4m1m7m"都是正确的。

**Q: 可以同时进行多个游戏吗？**

A: 每个群/频道同时只能有一个游戏。如果想开始新游戏，需要先用 `/清一色 stop` 停止当前游戏。

**Q: 连胜记录会保存吗？**

A: 连胜记录在插件运行期间保存，重启 AstrBot 后会清零。
